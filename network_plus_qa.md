# Network+ Practice Q&A

## Question 1
**Q:** What is the difference between a subnetting mask of `/24` and `/26`?  
**A:** A `/24` mask provides 256 IP addresses (254 usable hosts). A `/26` mask provides 64 IP addresses (62 usable hosts).  

## Question 2
**Q:** Which OSI layer is responsible for routing?  
**A:** The Network layer (Layer 3) handles routing.  

## Question 3
**Q:** What is the maximum cable length for 10BASE‑T Ethernet?  
**A:** 100 meters (328 feet).  

## Question 4
**Q:** How many bits are used for the host portion in a Class B address with a default subnet mask?  
**A:** 16 bits are used for the host portion.  

## Question 5
**Q:** What is the default administrative distance of a static route?  
**A:** 1.  

## Question 6
**Q:** Which command displays the routing table on a Windows machine?  
**A:** `route print` or `netstat -r`.  

## Question 7
**Q:** What is NAT overload (PAT)?  
**A:** Network Address Translation overload, also known as Port Address Translation, allows multiple private IP devices to share a single public IP by mapping each connection to a unique port.
